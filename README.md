# Will's Home Remodeling

Responsive one-page website preview for Will's Home Remodeling LLC in Palmyra, New Jersey.

## Preview notes

- Static HTML, CSS, and JavaScript
- Mobile click-to-call action
- Estimate form opens a prefilled email to the public business address
- Project photography is used with business approval and stored locally
- Search-engine indexing is disabled while this remains a preview

## Local preview

From the workspace root:

```bash
python3 -m http.server 18765 --bind 127.0.0.1
```

Open <http://127.0.0.1:18765/samples/will-home-remodeling-one-page/>.
